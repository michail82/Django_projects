Создание проекта django:      django-admin startproject name
Запуск сервера django:        python manage.py runserver 
Создание виртуальной среды:   virtualenv name
Активация виртуальной среды:  name/Scripts/activate
Выйти из виртуальной среды    deactivate
Посмотреть версию Django:     python -m django --version
Утсановть Django в среду:     pip3 install django==3.0.7
Создать APP в среде:          python manage.py startapp name
